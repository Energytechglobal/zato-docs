Using zato-2.0.7-trusty64 box
===============================

This box is pre-build with Zato 2.0.7 installation. Using our Vagrantfile
allows one to have Zato quckstart environment created and running.

Requirements
------------

* Vagrant
* VirtualBox

Usage instructions
------------------

* Download Vagrantfile

::

  $ wget Vagrantfile create_quickstart.sh

By default, the box's private IP address is set to 10.77.77.77, so Zato web admin
is available at that address at port 8183.

* Start the box after the download is complete.

::

  $ vagrant up

* ...and reload it when it's up already:

::

  $ vagrant reload

* Now you can SSH into the box:

::

  $ vagrant ssh

* While inside the box, you can switch to 'zato' user and check Zato version

::

  vagrant$ sudo su - zato
  zato$ zato --version
  Zato 2.0.7.rev-dfee180c
  zato$
